API Reference¶

Package Structure¶

The package structure is simple, with 5 main modules: io: , processing:, analysis, viz, and simulation. More information is available below

The Pycroscopy package.

Submodules¶

pycroscopy:

pycroscopy.analysis

Physical or chemical model-based analysis of data

pycroscopy.io

Translators to extract data from custom & proprietary microscope formats and write them to HDF5 files.

pycroscopy.processing

Physics-independent processing of data including machine learning, image processing, and signal filtering.

pycroscopy.viz

Plotting functions and interactive jupyter widgets for scientific applications
